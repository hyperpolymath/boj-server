= Bundle of Joy Server — Roadmap
:toc:

== Vision

A distributed, formally verified catalogue of developer server capabilities. AI goes to ONE place instead of hunting across dozens of MCP/LSP/etc servers. Community-hosted nodes share the load.

== Current Status: Phase 1 Complete

The core catalogue is built and verified. Three cartridges prove the pattern end-to-end.

== Milestones

=== v0.1.0 — Foundation (Phase 0+1) ✓

* [x] Fresh repo from rsr-template-repo (17 mandatory workflows)
* [x] Idris2 ABI: Catalogue, Protocol, Domain, Menu, Federation
* [x] `IsUnbreakable` proof type (only Ready cartridges can mount)
* [x] Zig FFI: catalogue mount/unmount with safety gate
* [x] C header: `boj_catalogue.h` from Idris2 encodings
* [x] Teranga menu (`menu.a2ml`) with three tiers
* [x] Order-ticket protocol (`order-ticket.scm`)
* [x] Three cartridges: database-mcp, nesy-mcp, fleet-mcp
* [x] Cultural respect documentation (Teranga/Umoja/Ayo)
* [x] Federation design (Umoja gossip, hash attestation)

=== v0.2.0 — Live Cartridges (Phase 2-3)
